useRef의 기능은
1. 리액트 DOM 수정
2. 리렌더링 되지 않는 내부 변수

리액트의 useState 값을 비롯한 const, let으로 선언한 변수는
리액트가 렌더링될 때마다 초기화값으로 지정된 값으로 초기화된다.
그에 반해 

const num = useRef(1);
로 선언한 num 변수는 num.current += 1;를 이용하여 관리하면
리렌더링 되더라도 1로 초기화되지 않고 기존의 값을 유지한다.
num이라는 객체의 안에 있는 current 값이 변경된 것이기 때문에
업데이트된 상태를 유지할 수 있는 것이다.

const num = {current : 1};
의 일반 객체 변수의 경우 1로 초기화되기 때문에 사용하지 않는다.
num.current를 호출할 때마다 1로 초기화가 된다.
